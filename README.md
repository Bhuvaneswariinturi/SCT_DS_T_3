# SCT_DS_Task_3

## Decision Tree Classifier for Bank Marketing Dataset

This project was completed as part of the Data Science Internship at SkillCraft Technology.

### Objective
Build a Decision Tree Classifier to predict whether a customer will subscribe to a product or service based on demographic and behavioral data.

### Dataset
Bank Marketing Dataset from the UCI Machine Learning Repository.

### Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

### Project Workflow
1. Data Loading
2. Data Preprocessing
3. Label Encoding
4. Train-Test Split
5. Decision Tree Model Training
6. Prediction and Evaluation
7. Decision Tree Visualization

### Results
The Decision Tree model was trained and evaluated using the Bank Marketing Dataset. The model predicts customer subscription behavior and demonstrates the application of machine learning classification techniques.
